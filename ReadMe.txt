How to Open and Run This Project in Google Colab
1. Open the Notebook

You can launch the .ipynb notebook directly in Google Colab:

Go to your GitHub repository

Click on the notebook file (e.g., wine_analysis.ipynb)

Click “Open in Colab” (top center)

If the button does not appear:

Right-click the notebook → Copy link

Open https://colab.research.google.com

Select GitHub tab

Paste your repo URL and open the notebook

Running the Notebook

Once inside Colab:

Step 1 — Ensure the Runtime Is Ready

Go to Runtime → Change runtime type

Set:

Runtime type: Python 3

Hardware accelerator: None (CPU is enough)

Step 2 — Run Each Cell

Run each cell in order by pressing:

Shift + Enter

You will see:

Data loaded successfully
